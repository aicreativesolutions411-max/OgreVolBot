# Swamp Sprite Prompt Pack — paste into any free image generator

Use a FREE generator that does DALL·E 3 / good art:
- **Bing Image Creator** (bing.com/create) — free, DALL·E 3, fast.
- **Leonardo.ai** — 150 free credits/day, has game-asset models + transparent PNG.
- **Ideogram / Krea / Playground** — free daily tiers.

Each prompt asks for a **solid magenta (#FF00FF) background** so the Sprite Forge
import tool can cut it to transparent automatically. Save each image with the
filename in **bold** (e.g. `ogre-boss.png`) — then drag them onto
`slimewire.org/sprite-forge.html` (Import mode) → Download All (.zip) → unzip into
`web/public/assets/slimewire/swamp/` → tell Claude to wire + push.

Shared style suffix (already included in each line below):
> Painterly realistic dark-fantasy game character, Warcraft-movie quality, soft volumetric shading, cool moonlit rim light from top-left and warm fire glow from below, full body centered with margin, on a SOLID FLAT PURE MAGENTA #FF00FF background, no scenery, no text, no shadow on background.
