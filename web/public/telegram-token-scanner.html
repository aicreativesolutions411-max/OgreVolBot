FAQ

What is a Telegram token scanner?

A Telegram token scanner lets users paste a contract address in chat and receive token context, chart links, warnings, and next actions.

Can groups use a token scanner for launches?

Yes. Group scanners can help launch rooms and communities share cleaner token context and chart links.

Should Telegram scan cards include warnings?

Yes. Scan cards should make risk visible and should not imply a token is safe just because it was scanned.
