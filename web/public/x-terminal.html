FAQ

Can X host a full SlimeWire trading terminal inside a post?

No. A normal X post cannot host a full embedded web app, but a pinned post can act as a command hub that routes users to scan replies, chart links, the web terminal, and the Telegram bot.

Can users trade directly from X?

Not from a normal X post. Linked users can stage a buy or sell from the live X DM terminal or its mobile Trade Pad, then reply YES or NO in the same DM before anything executes.

What should users tag on X?

Useful public commands include scan plus a contract address, chart plus a contract address, and rug plus a contract address. Full swaps, bundles, TP/SL, launch tools, and wallets live in the web terminal and Telegram bot.

Does SlimeWire ask for seed phrases in X DMs?

No. SlimeWire should never ask for seed phrases or private keys in X, Telegram, or the web terminal.

Why pin this page on X?

A pinned terminal page gives visitors one obvious command center for scans, charts, proof, Telegram, and wallet-approved trade handoff without cluttering every reply.
