FAQ

How do I add SlimeWire Bot to a Telegram group?

Open the add-to-group link, choose the group, approve Telegram's bot invite flow, then use SlimeWire commands or group setup options inside the chat.

What can @SlimeWiredBot do in a crypto group?

@SlimeWiredBot supports SlimeWire workflows such as Solana token scans, buy alerts, raid cards, launch context, proof links, and handoff to the SlimeWire terminal.

Does the bot need admin permissions?

Some group features may need admin permissions, but basic bot access should stay limited to the features the group wants to use.

Will the bot ask for my seed phrase?

No. A legitimate SlimeWire flow should never ask for a seed phrase or private key in Telegram.
