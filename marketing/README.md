# SlimeWire Marketing / Listing Pack

This folder is for free listing, directory submission, and outreach prep.

## Files

- `listing-submission-kit.md` - canonical copy, links, descriptions, categories, tags, and safety language.
- `free-listing-tracker.csv` - prioritized target list for free directories, communities, listicles, and ecosystem pages.
- `outreach-templates.md` - copy-paste templates for directories, Telegram bot lists, Solana ecosystem pages, and listicle writers.
- `submission-playbook.md` - operating process for submitting without spammy or risky behavior.

## Fast Use

1. Open `free-listing-tracker.csv`.
2. Start with `priority=1`.
3. Pull copy from `listing-submission-kit.md`.
4. Use `outreach-templates.md` if the site requires a message.
5. Update `status`, submission date, notes, and live listing URL.

Do not submit fake reviews, make guaranteed-profit claims, or use suspicious listing sites that ask for wallet seed phrases.
